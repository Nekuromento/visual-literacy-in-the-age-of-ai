# Visual Literacy In The Age Of AI

Interactive archival version of the closed Tally survey at:

https://tally.so/r/dWzZJV

This static site preserves the educational image-detection experience, training material, solution images, and authorship note. It omits demographic, consent, and data-collection inputs, and it does not submit responses.
